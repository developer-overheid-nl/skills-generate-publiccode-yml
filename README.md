# Generate publiccode.yml skill

This Agent Skill can be used to generate a publiccode.yml file for your project.

It is our intention to internationally co-operate on improving this skill to
make it usable for all European member states 🇪🇺

## Agent Skills standard

This project adheres to the [Agent Skills](https://agentskills.io/home) format,
a standardized way to give AI agents specialized knowledge and workflows.

## Projects using this skill

The following projects had their `publiccode.yml` generated by this skill:

- [RIVM-bioinformatics/Pacini-typing](https://github.com/RIVM-bioinformatics/Pacini-typing/blob/main/publiccode.yml)
- [developer-overheid-nl/don-site](https://github.com/developer-overheid-nl/don-site/blob/main/publiccode.yml)
